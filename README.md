# OSCC CapsuleNet XAI

## Explainable Deep Learning Framework for Oral Squamous Cell Carcinoma Histopathology Classification Using Capsule Networks

This repository presents an Explainable AI-driven deep learning framework for automated classification of Oral Squamous Cell Carcinoma (OSCC) histopathology images using Capsule Networks and MobileNet-based architectures.

The framework integrates multiple Explainable AI (XAI) techniques including:
- Capsule Activation Vectors
- LIME
- Occlusion Sensitivity Mapping

to improve interpretability and transparency of histopathological image classification.

---

# Overview

Oral Squamous Cell Carcinoma (OSCC) is one of the most prevalent forms of oral cancer and represents a major global health burden. Histopathological examination remains the gold standard for diagnosis; however:
- visual interpretation is time-consuming
- subtle tissue variations may lead to inter-observer variability
- diagnostic consistency remains challenging

This work proposes an automated deep learning framework for OSCC histopathology classification using Capsule Networks and lightweight CNN architectures.

The framework aims to:
- improve automated cancer classification
- preserve spatial feature relationships
- enhance interpretability using Explainable AI
- support computer-aided pathology systems

# Explainable AI Techniques

## Capsule Activation Vectors
Analyzes capsule activations to understand feature representations learned by the Capsule Network architecture.

## LIME
Provides local interpretable explanations highlighting image regions contributing to predictions.

## Occlusion Sensitivity Mapping
Measures prediction sensitivity by systematically masking image regions to identify diagnostically important tissue structures.
